Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Usamos a tag CODE para (além de formatar o texto em pequeno fragmento de código de programação) informar uma semântica de conteúdo em código de programação para o navegador.

    
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Nota: usamos a tag PRE para pré formatar o texto, ou seja, o que escrevemos no código fonte do HTML aparece na tela do navegador, assim conseguimos deixar o código acima aparecendo na tela igual conforme escrevemos um código de programação.

Segundo Jeff Noble, no seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto. Este é um exemplo de citação deslocada em blockquote.

Usamos a tag BLOCKQUOTE para inserir uma citação deslocada e usando parâmetro cite="citacao-de-exemplo.com" podemos criar uma referência para citação, uma referência somente para os mecanismos de buscas (não cria um link para citação, mas criamos somente um referência semântica para manter os direitos de uso do autor).
