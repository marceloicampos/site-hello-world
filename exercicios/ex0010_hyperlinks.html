Usando links externos

Usamos a TAG <a> para criar Hyperlinks ou Âncoras.

  • O PARÂMETRO href="https://www.xxx.com.br" define a URL que a TAG <a> aponta.
  • O PARÂMETRO target="_blank" define o alvo do Hyperlink com os valores _blank para abrir em nova aba/página ou _self para abrir na mesma aba/página.
  • O PARÂMETRO rel="external" define o tipo de Hyperlink como o valor external para relativizar o Hyperlink como externo, next indica que o link é para a próxima parte do documento atual, prev indica que o link é para a parte anterior do documento atual e author indica que é um link para o site do autor do artigo atual.
  • O PARÂMETRO hreflang="en" indica o tipo de idioma do Hyperlink com os valores, por exemplo, en para lingua inglesa.

Você pode me enviar um e-mail por aqui que irá abir o programa de e-mail padrão para enviar e-mail.

Você pode acessar o meu repositório público no GitHub usamos o target _self que irá abrir na mesma aba/página.

Você também pode acessar o meu canal no YouTube usamos o target _blank que irá abrir em outra aba/página.
