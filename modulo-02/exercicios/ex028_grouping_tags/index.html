Grupo de Tags Semânticas em HTML5

As Semantic Grouping Tags deixam o web site organizado de forma semântica, ou seja, separamos dentro do HTML o cabeçalho (header) da parte principal (main), deixamos a parte principal (main) separada em parte como sessões (section), artigos (article), laterais (aside) e por fim o rodapé (footer). Esta separação conta muito para o SEO e o ranking dos crawlers (como o buscador do Google)
