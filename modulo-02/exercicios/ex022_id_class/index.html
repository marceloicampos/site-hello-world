PODEMOS DEFINIR SOMENTE 1 ELEMENTO POR ID, ou seja, se definirmos um id para um elemento só podemos ter aquele id para aquele elemento, a ligação de id e elemento é única entre eles.

Aprendendo CSS3

CSS3 Básico

CLASS chama o PONTO

CSS3 Intermediário

Em HTML usamos class=" " e em CSS é .

CSS3 Avançado

PODEMOS DEFINIR VÁRIOS ELEMENTOS PARA MESMA CLASS, ou seja, vários elementos de um HTML podem ser definidos para mesma class.

PARA UMA BOA PRÁTICA: devemos dar nomes para id e class não de acordo com a forma a ser aplicada, mas de acordo com a função que elas exercem no projeto como um todo.

NOTA: podemos aplicar mais de uma class para um mesmo elemento, porém podemos aplicar somente um único id para um elemento, ao contrário podemos aplicar diversas class para um mesmo elemento, porém o id sempre sobrepõem as configurações da class.

NOTA: primeiro configurações genéricas, depois os id e depois as class.
