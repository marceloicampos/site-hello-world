  • A tag div é um elemento de divisão que gera um container para conteúdo de fluxo, que de certa forma não representa nada, mas serve para efetuar divisões no HTML.
  • A div pode ser utilizado para agrupar elementos para fins de estilos usando class ou id, ou porque eles compartilham valores de atributos como lang para com linguas diferentes.
  • A div deve ser utilizada somente quando não tiver outros elementos de semântica tal como article ou nav, pois as divisões main, article, aside e ou nav dividem semanticamente a página.
