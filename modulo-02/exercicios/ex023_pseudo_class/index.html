Trabalhando com Pseudo Classes

  • # é id
  • . é class
  • : é pseudo classe, onde essas devem estar relacionadas ao estado de um elemento ou de uma class, no exemplo seguir usamos o hover e ao passar o mouse por cima a cor muda para amarelo.

A tag div é um elemento de divisão que gera um container para conteúdo de flux, que de certa forma não representa nada.
Ele pode ser utilizado para agrupar elementos para fins de estilos usando class ou id, ou porque eles compartilham valores de atributos, como lang.
Ele deve ser utilizado somente quando não tiver outro elemento de semântica tal como article ou nav.
