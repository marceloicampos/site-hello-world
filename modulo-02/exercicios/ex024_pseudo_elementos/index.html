  • CHILDREN leva > como exemplo: div > p, onde só vamos formatar o p da div.

  • PSEUDO CLASSES leva : exemplo: a:hover, onde ao passar o mouse mudamos o estado do elemento.

  • PSEUDO ELEMENTOS leva :: exemplo a::before, onde podemos incluir conteúdo ao elemento antes do elemento.


- VERDE É O LINK NÃO VISITADO 'a'

- VERMELHO É O LINK VISITADO 'a:visited'

- PRETO É O LINK APONTADO COM O MOUSE 'a:hover'

- AMARELO É O LINK ATIVADO PELO MOUSE 'a:active'

- O LINK MEU SITE ESTÁ TRATADO COM PSEUDO ELEMENTO

- O PSEUDO ELEMENTO TRABALHA COM O CONTEÚDO PERIFÉRICO DOS ELEMENTOS
