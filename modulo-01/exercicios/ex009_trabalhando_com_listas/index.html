Listas Ordenadas - OL

  1. a TAG <ol> cria Lista Ordenadas.
  2. o PARÂMETRO type mostra a lista por tipos que podem ser:
  3. 1 (números - PADRÃO); A (letras maiúsculas) ; a (letras minúsculas); I (algarismos romanos maiúsculos); i (algarismos romanos minúsculos).
  4. o PARÂMETRO start indica onde a ordem da lista deve começar, nesse caso indiquei o início em 3.
  5. os itens de listas são criados pela TAG <li >.
  6. NOTA: a partir do html5 o fechamento dos items de listas (pela TAG <li>) são opcionais.
  7. NOTA: na boa prática devemos indicar o PARÂMETRO type na CSS usando a propriedade list-style-type, pois as formatações são representadas dentro da CSS. Esta regra vale tanto para as OL como para UL.

Listas não Ordenadas - UL

  • a TAG <ul> cria Lista Não Ordenadas.
  • o PARÂMETRO type mostra lista por tipos que podem ser:
  • disc (PADRÃO); circle ; square, no caso usei o disc.
  • NOTA: geralmente os menus dos sites são criados por listas não ordenadas. MUITO IMPORTANTE SABER ESSE TIPO DE CONSTRUÇÃO.
