Site em HTML5 com Semântica

Em HTML5 o significado de algumas TAGs tem mais relevância que a forma, ou seja, a Tag b que representa somente a forma em negrito está em desuso (e NÃO devemos mais usar), então atualmente com o HTML5 usamos tags com significado (ou valor semântico) usando a Tag strong que além de inserir a forma de negrito impõe um valor ao texto de "algo forte" para o conteúdo.

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica então não faz sentido usar a tag B).

Nesta frase, tenho um termo em destaque usando a tag STRONG (com semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag i (não semântica então não faz sentido usar tag i).

Nesta frase, temos um termo em ênfase usando a tag em (com semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag mark.

E neste outro parágrafo, temos outro texto marcado com parâmetro style em css dentro da própria tag mark para mudar a cor do mark, do texto que está marcado e peso da fonte para super negrito.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. NOTA: tag big é obsoleta, porém a tag small ainda é atual, pois a exemplo podemos usar letra pequena com semântica para indicar uma nota de rodapé.

Texto apagado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag del (semântica).

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag ins (semântica).

Existe também o sublinhado com a tag u (não semântica então não faz sentido usar a tag u).

Texto sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag sup.

Texto subscrito

Para inserir coisas do tipo H2O usamos a tag sub
