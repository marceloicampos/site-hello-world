Notas de favicon: PERCEBA QUE O FAVICON DESSA PÁGINA É DIFERENTE DAS OUTRAS PÁGINAS

  1. Podemos colocar outros Favicon para outras páginas específicas do website;

  2. Usamos um arquivo com extensão .ico para inserir os Favicon;

  3. De preferência use o nome de padrão de arquivo = favicon.ico;

  4. Para inserir um favicon usamos a TAG <link> com parâmetros rel="shortcut icon" >>> href="./favicon.ico" >>> type="image/x-icon" dentro da TAG comportamental <head> ;

  5. Escrevemos assim no html: <link rel="shortcut icon" href="./favicon.ico" type="image/x-icon"> ;

  6. Dica: o Visual Studio Code tem atalho "link:favicon" para criar a linha acima

  7. Podemos copiar um favicon da internet usando por exemplo o site: Icon Archive, ou criar o próprio favicon usando por exemplo o site: favicon.ico Generator ou ainda transformar um arquivo de texto, png ou emoji usando por exemplo o site: The Best Favicon Generator ;

  8. Utilize da sua criatividade para criar o favicon do seu website.
