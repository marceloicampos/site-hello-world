Capítulo 1

Capítulo 1.1

CSS ou Cascading Style Sheets ou Folhas de Estilo em Cascata é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML (ou simplesmente formatação de conteúdos). A recomendação atual é a CSS3.

Capítulo 1.2

Este documento contém códigos em CSS In Line, uma codificação não recomendada para CSS, pois mistura códigos HTML com código em CSS num mesmo documento. Além termos que repetir várias vezes códigos CSS não damos uma organização separada para os códigos.

Capítulo 2

Capítulo 2.1

Prefira então CSS interno no HEAD da página ou CSS externos via link no HEAD. Nota: códigos In Line irão sobrepor os códigos internos que irão sobrepor códigos externo, então os códigos In Line podem servir para pequenas correções de CSS, mas não é recomendado.
